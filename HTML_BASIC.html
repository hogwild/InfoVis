some basics of HTML

  1. Elements are the basics of HTML. Different elements are represented by different tags. For example, "<p> </p>" is the tag of element "paragaraph"; "<p>" is called opening tag and "</p>" is called closing tag
  2. Every element has its own attributions which can be set in the opening tag. For example, in <p lang="en-us">, "lang" is the attribution of language. Please note, even the events that related to the element can also be set in this way. For example, in < button type="submit" id="btn_submit" onclick="btnAction()">, onclick is one of the events belonging to button, it can be connectted to the js function bitAction() in the opening tag. However, you are not encouraged to connected js function in this way, because it makes the HTML and js code mixed together. A better solution is to do it in a js file only.
